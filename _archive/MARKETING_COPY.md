## Notes for integration

- Strings are written to be drop-in for `i18n` files. Suggested keys mirror the section headings (e.g. `tutorial.step1`, `gameover.range.warmup`, `push.streak.reminder`).
- No CSS variables or class names are introduced here — copy only. The pack does **not** redefine `--color-primary` or `--color-bg` from `styles.css`, and does **not** depend on `Board.js` internals; it only references public concepts (rows, columns, 3×3 squares, hint, undo, modes) consistent with `getCell` / `setCell` / `clearLine` semantics.
- Char counts on every length-bound field were checked against the stated limits.
